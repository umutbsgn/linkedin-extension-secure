LinkedIn Extension Secure API

API Endpoints

  • /api/healthcheck - Check if the API is working
  • /api/anthropic/analyze - Analyze text using Anthropic API
  • /api/analytics/track - Track events in PostHog
  • /api/supabase/auth/login - Login with Supabase
  • /api/supabase/auth/signup - Signup with Supabase
  • /api/supabase/user-settings - Manage user settings
  • /api/supabase/beta-access - Check beta access

Status

This API is used by the LinkedIn Extension Chrome extension to securely handle API calls to Anthropic, Supabase, and PostHog.

All API keys are stored securely in Supabase and are never exposed in client-side code.
