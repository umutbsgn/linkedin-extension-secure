API Endpoint Tests

Authentifizierung

Wichtig: Für die meisten Tests wird ein gültiger Authentifizierungstoken benötigt.

  • Authentifizierungstoken generieren
    Melden Sie sich an oder registrieren Sie sich, um einen Authentifizierungstoken zu erhalten

Vercel-Verbindungstests

  • Vercel-Verbindung testen (einfache Version)
    Überprüft die Verbindung zum Vercel-Backend ohne ES-Module (funktioniert direkt im Dateisystem)
  • Vercel-Verbindung testen (mit ES-Modulen)
    Überprüft die Verbindung zum Vercel-Backend mit ES-Modulen (benötigt einen lokalen Server)

Supabase-Konfigurationstests

  • Supabase-Konfiguration testen
    Überprüft die Supabase-Konfiguration (URL und API-Schlüssel)

API-Endpunkttests

  • Alle API-Endpunkte testen
    Umfassende Tests für verschiedene API-Endpunkte (PostHog, API-Nutzung, Anthropic)

Supabase RPC-Funktionstests

  • RPC-Funktionen testen (Node.js-Skript)
    Führen Sie das folgende Kommando in der Konsole aus:
    node test-rpc-functions.js <user_id>
    Hinweis: Benötigt die Umgebungsvariablen SUPABASE_URL und SUPABASE_SERVICE_KEY
