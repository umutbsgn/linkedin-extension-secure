How to get a Supabase Authentication Token:

  1. Open the browser extension
  2. Log in to your account
  3. Open the browser's developer tools (F12 or right-click > Inspect)
  4. Go to the "Application" tab
  5. In the left sidebar, expand "Local Storage"
  6. Click on the extension's domain
  7. Look for a key named "supabaseAuthToken" or "sb-[project-ref]-auth-token"
  8. Copy the value (it's a long string that starts with "ey...")
  9. Paste it in the field above

Note: The token is temporary and will expire after some time.
